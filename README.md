# 售票資訊整合

這是整合後的手機 App 專題資料夾。

## 資料夾內容

- `mobile-app`：Android 原生手機 App，使用 Kotlin + Jetpack Compose。
- `backend`：Node.js + Express 後端 API，連接 MySQL 資料庫。
- `啟動後端.bat`：啟動 API 伺服器。
- `建置APK.bat`：重新建置 Android APK。
- `專題展示說明.md`：展示流程與給老師看的說明。

## 快速展示

1. 雙擊 `啟動後端.bat`。
2. 用 Android Studio 開啟 `mobile-app`。
3. 啟動 Android 模擬器並按 Run。
4. 依序展示首頁、搜尋、提醒、分析。

## APK

已建置的 APK 在：

```text
mobile-app\app\build\outputs\apk\debug\app-debug.apk
```
